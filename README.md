# design_tela_selecao

Protótipo em React + Vite para simular a experiência de candidatura em um processo seletivo.

## O que foi implementado

- painel inicial do candidato com cards de etapas
- fluxo detalhado de "Dados de Inscrição" com progresso por etapas
- formulário dividido em blocos menores para reduzir rolagem excessiva
- sidebar com status da inscrição e acompanhamento do progresso
- interface construída com `react-bootstrap`

## Executar localmente

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
```
